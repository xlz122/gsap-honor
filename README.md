## 环境要求

需要 NodeJS 20.0.0+ 环境

## 项目简介

本项目是一个仿写[荣耀官网首页](https://xieyufei.com/2023/04/14/GSAP-Practise.html)项目

## 项目使用技术栈

* React
* Gsap
* TypeScript

## 运行

启动项目

```
yarn install
```

```
yarn dev
```

打包项目

```
yarn build
```
